# SIMS User Guide

## Introduction
The Student Information Management System (SIMS) allows students to manage their academic profile, view courses, and check grades.

## Getting Started

### Login
1. Navigate to the login page.
2. Enter your Student ID (e.g., `ETS0456/13`).
3. Enter your password (default: `password123`).
4. Click "Login".

### Dashboard
After logging in, you will be directed to the Dashboard. Here you can see a quick overview of available modules.

## FAQs

**Q: How do I change my password?**
A: Currently, password changes are handled by the system administrator.

**Q: What if I see a missing grade?**
A: Please contact the registrar office if a grade is missing after the publication date.
